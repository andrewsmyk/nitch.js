nitch.events.visibly

new nitch.events.visibly

Wrapper supporting the Page Visibility API, which will fire events when people hide the window (e.g. change tabs because the boss is coming). We don't support the "prerender" visibility state.

Parameters:

Name Type Description
opts Object
Properties
Name Type Description
onVisible Function Fires when the page becomes visible
onHidden Function Fires when the page becomes visible

Source: nitch.js, line 436

Example

 nitch.events.visibly({
onVisible: function() { yourGame.restart(); },
onHidden: function() { yourGame.pause(); }
});
