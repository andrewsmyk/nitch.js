timing

new nitch.debug.timing

Wrapper for the Navigation timing API to help you debug your game

Parameters:

Name Type Description
opts Object
Properties
Name Type Description
list Boolean List print out the performance timing and navigation results in a definition list at the end of the <body>
timeline String Generate a timeline like Firebug's network tab or Chrome's timeline panel after the element specified
