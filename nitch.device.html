nitch.device.setAppcache <static>

Injects an appcache manifest based on UserAgent and if it's a mobile, with a media query

Parameters:

Name Type Description
appcache Object
Properties
Name Type Description
device String Device name with the url of the manifest
url String Url of the manifest.
For desktop and tv support just use the device name 'desktop' or 'tv'.

Example

 nitch.device.appcache({
	desktop: "style/desktop/appcache.manifest",

	iphone4s: "style/iphone4s/appcache.manifest",

});

nitch.device.watchOrientation <static>

Adds a class to the body and an event to update when orientation changes for the iPhone 3 (iOS 4.2.1), as it doesn't support media queries in CSS.
This is called by default when the document is ready
