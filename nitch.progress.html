nitch.progress

new nitch.progress

Create a progress bar on screen and update it as you see fit. The progress bar also applies the WAI Aria progress bar role by default and updates aria-valuenow as you send loaded assets.

Parameters:

Name Type Description
element String Element to attach the progress bar to
opts Object
Properties
Name Type Argument Default Description
defaultLoaded Numeric <optional>
 0 The default amount loaded when the progress bar starts
assets Object Object to specific your assets and the percentage of how much they are loaded
onComplete Function Function to call once the total of the assets has been reached.

Source: progress.js, line 1

See:
  • Simple example

Example

 var loader = new nitch.progress("#loading", {
	defaultLoaded: 40,
	assets: {
		audio: 20,
		video: 20,
		graphics: 20
	},
	onComplete: function() { console.info("I'm done loading"); }
});

// Then fire your loaded events
loader.loaded('audio');
loader.loaded('video');
loader.loaded('graphics');

nitch.progress.failed <static>

Replaces the progress bar with a error message should you need to stop loading things. An example usage might be as part of the loading process you want to check browser support.

Parameters:

Name Type Description
message String The failure message you want to display to the user
